canvas标签

用来画图的标签

获得画图对象的方法

canvas标签.getContext（‘2d’） 获得一个2d对象

画一条线

*context*.arc(x,y,r,sAngle,eAngle,counterclockwise);

beginPath()

closePath()

创建线性渐变

参数 描述
stop 介于 0.0 与 1.0 之间的值，表示渐变中开始与结束之间的位置。
color 在 stop 位置显示的 CSS 颜色值。

指定线性渐变填充

创建放射渐变

save() 保存当前的信息

restore() 回到上次保存的存档

drawimage()对引入图片的剪切缩放，会按缩放的比例缩放，

例如·50到25，会缩小一半

参数 描述
img 规定要使用的图像、画布或视频。
sx 可选。开始剪切的 x 坐标位置。
sy 可选。开始剪切的 y 坐标位置。
swidth 可选。被剪切图像的宽度。
sheight 可选。被剪切图像的高度。
x 在画布上放置图像的 x 坐标位置。
y 在画布上放置图像的 y 坐标位置。
width 可选。要使用的图像的宽度。（伸展或缩小图像）
height 可选。要使用的图像的高度。（伸展或缩小图像）

文本

font属性 设置字体有关的属性

textAlign 对其方式 star left 水平靠左

textbaseline 垂直方向

填充文本 fillText(文本，x,y) //x y是坐标

描文本线 strikeText()

图像绘制

方法 描述
drawImage() 向画布上绘制图像、画布或视频。

像素操作

属性 描述
width 返回 ImageData 对象的宽度。
height 返回 ImageData 对象的高度。
data 返回一个对象，其包含指定的 ImageData 对象的图像数据。

data对象储存的是每个像素点储存的是rgba颜色

getimageDate 方法返回的是一个一维数组，里面放的是data数据
