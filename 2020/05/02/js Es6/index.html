变量/赋值

  1. var 可以重复赋值 修改 没有块级作用域
  2. let 变量 不可重复赋值 有块级作用域
  3. const 常量 不可重复赋值 不可修改

  • 解构赋值

    let [a,b,c,d] = [1,2,3,4]

    let {a,b,c,d} = {a:1,b:2,c:3,d:4}

    两个注意点

    1. 左右两边一致，得是个东西 里 [] = [] {} = {}
    2. 定义赋值必须同步完成

函数

1.箭头函数

  • function关键字可以省略，参数用（）括起来
  • 当且仅当 只有一个参数时，（）可以省略，仅有一条语句而且是return时，{}可以省略

2.默认参数

如果没有参数传进来。则有默认参数

function(a=1){}
