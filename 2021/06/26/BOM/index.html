window对象

属性

  • window.name

    • 表示当前浏览器窗口的名字

  • window.closed

    • 返回布尔值，表示窗口是否关闭

  • window.opener

    • 表示当前窗口的父窗口，如果没有返回null

  • window.self

    • window.window

      • 都指向窗口本身

  • window.frames

    • 返回页内所有框架窗口

  • window.frameElement

  • window.top

    • 指向最顶层窗口

  • window.parent

    • 指向父窗口

  • window.devicePixelRatio

    • 返回一个比率

  • 位置大小(只读)

    • 浏览器左上角相对于屏幕左上角的距离

      • window.screenX
      • window.screenY

    • 返回网页在当前窗口的可见部分

      • window.innerHeight
      • window.innerWidth

    • 返回浏览器的总高度，宽度

      • window.outerHeight
      • window.outerWidth

    • 返回页面滚动距离

      • window.scrollX

        • ==

          • window.pageXOffset

      • window.scrollY

        • ==

          • window.pageYOffset

  • 浏览器组件

    • window.locationbar

      • 地址栏对象

    • window.menubar

      • 菜单栏对象

    • window.scrollbars

      • 窗口的滚动条对象

    • window.toolbar

      • 工具栏对象

    • window.statusbar

      • 状态栏对象

    • window.personalbar

      • 用户安装的个人工具栏对象

    • 组件共同属性

      • visible

        • 表示这些组件是否可见

  • 全局对象属性

    • window.document

      • document对象

    • window.location

      • 指向location对象

    • window.localStorage

      • 指向本地储存的localStorage数据

    • window.sessionStorage

      • 指向本地储存的sessionStorage数据

    • window.console

      • 指向console对象，用于操作控制台

    • 指向Screen对象

      • 表示屏幕对象

  • window.isSecureContext

    • 返回布尔值，https协议下返回true

方法

  • 对话框

    • window.alert

    • window.prompt(tips,defaultValue)

      • 用户输入内容，有确定和取消，确定返回输入，取消返回null

    • window.confirm(tips)

      • 有确定和删除按钮，确定返回true,取消返回false

  • 窗口相关

    • window,open(url,windowName,[features])

      • features逗号分割的键值对,可以设置组件工具是否显示

    • window.close()

      • 关闭窗口

    • window.stop()

      • 停止加载

  • 窗口移动

    • window.moveTo(x,y)

      • 移到指定位置

    • window.moveBy(x,y)

      • 移到相对位置x,y

  • 窗口缩放

    • window.resizeTo(width,height)

      • 缩小到指定大小

    • window.resizeBy(x,y)

      • 按照相对量进行缩放

        • 负数就表示当前窗口大小减去该值

  • 窗口滚动

    • window.scroll

      • ==

        • window.scrollTo()

          • 页面滚到指定位置

          • 接受两个参数或一个参数对象

            • 两个参数

              • x,y坐标

            • 参数对象

              • top：y坐标

              • left:x坐标

              • behavior:滚动方式

                • smooth
                • instant
                • auto

    • window.scrollBy(右，下)

      • 网页滚动到指定距离

    • 滚动某个元素

      • scrollTop
      • scrollLeft
      • scrollIntoView

  • window.print

    • 跳出打印对话框

  • 窗口焦点

    • window.focus()

      • 聚焦

    • window,blur()

      • 失焦

  • window.getSelection()

    • 返回Selection对象，表示用户现在选中的文本

  • window.getComputedStyle()

  • window.matchMedia()

    • 检查媒体查询

  • window.requestAnimationFrame()

  • window.requestIdIeCallback()

Navigator对象

属性

  • Navigator.userAgent

    • 浏览器标识

  • Navigator.plugins

    • 返回类数组，浏览器安装的插件

  • Navigator.platform

    • 用户的操作系统信息

  • Navigator.onLine

    • 返回布尔值，用户在线还是离线(判断网络)
    • 在线触发online事件
    • 离线触发offline事件

  • Navigator.language

    • 返回字符串，标识浏览器首选语言，属性只读

    • Navigator.languages

      • 返回一个数组，表示用户用户可以接受的语言

  • Geolocation

    • Navigator.geolocation()返回该对象(https支持)

    • Geolocation.getCurrentPosition()

      • 得到用户当前位置

    • Geolocation.watchPositon()

      • 监听用户位置变化

    • Geolocation.clearWatch()

      • 取消watchPosition方法指定的函数

  • Navigator.cookieEnabled

    • 返回一个布尔值，表示浏览器Cookie功能是否打开

  • 实验性属性

    • Navigator.deviceMemory

      • 返回当前计算机的内存数量，四舍五入返回最接近2的幂

    • Navigator.hardwareConcurrency

      • 返回可用的逻辑处理器的数量，只读
      • 该属性通过用于Web Worker,每个可用的逻辑处理器都创建一个Worker

    • Navigator.connection

      • 返回一个对象，包含当前对象

方法

  • Navigator.javaEnable()

    • 返回一个布尔值，表示浏览器能否能运行Java Applet小程序

  • Navigator.sendBeacon()

    • 用于向服务器异步发送数据

screen.colorDepth

screen.orientation

AJAX

XMLHttpRequest() 返回一个实例

xhr实例

  • 属性

    • readyState 返回一个整数，表示实例对象当前状态

    • readyState

      • 0

        • 实例以已生成，open方法没有调用

      • 1

        • open方法已调用，send未调用

      • 2

        • send已调用，收到返回的头信息和状态码

      • 3

        • 正在接收服务器传来的数据体(body部分)

      • 4

        • 表示服务器返回的数据已经完全接受，或本次接受已经失败

    • xhr.onreadystatechange,readyState发生改变时触发该属性指向的函数

    • xhr.response 服务器返回的数据，类型右xhr.responseType属性决定

    • xhr.responseType 在open方法后，send方法前调用

      • “”

        • 返回文本

      • arraybuffer

        • 服务器返回二进制数组

      • blob

        • 返回二进制对象

      • document

        • 返回一个文档对象

      • json

        • JSON对象

      • text

        • 字符串

    • xhr.responseText

      • 从服务器接收到字符串

    • xhr.responseXML

      • HTML或XML文档对象

    • xhr.responseURL

      • 返回目标网址

    • xhr.status

      • http状态码

    • xhr.statueText

      • 返回状态提示，默认ok

    • xhr.timeout

      • 设置请求超时的标准，单位为毫秒

    • xhr.ontimeout

      • 超时监听属性，超时会触发

    • xhr.upload

      • xhr上传文件时会有这个对象

  • 事件的监听属性

    • onloadstart

      • 请求发出时触发

    • onprogress

      • 正在发送和加载数据时触发

    • onabort

      • 请求终止时触发，比如调用了abort方法

    • onerror

      • 请求失败触发

    • onload

      • 请求成功触发

    • ontimeout

      • 超时触发

    • onloadend

      • 请求完成触发，不论成功与否

  • 方法

    • xhr.open(method,url,async,user,password)

    • xhr.send() 发送http请求

    • xhr.setRequestHeader()

      • 设置请求头

    • xhr.getResponseHeader(filedname)

      • 获取响应头某一字段的值

    • xhr.getAllResponseHeader()

      • 获取所有响应头

    • xhr.abort()

      • 终止请求

    • xhr.sendBeacon()

Storage

window.sessionStorage和window.localStorage

  • 只是保存时长不一样，接口一样

key-value形式存储

唯一属性

  • length

    • 返回保存项的个数

方法

  • setItem(key,value)

    • 存入数据

      • 两个参数都是字符串，不是字符串会转成字符串
      • 存对象得先转成字符串

  • getItem(key)

    • 根据key取value，key不存在返回null

  • removeItem(key)

    • 根据key删除

  • clear()

    • 清除所有保存的

  • key(num)

    • 根据下标返回对应的项

事件

  • storage储存的数据发生改变时会触发window.storage事件

History

window.history 表示当前窗口的浏览历史

属性

  • history.length

    • 当前窗口访问过的网址的对象

  • history.state

    • 最上层的状态值

方法

  • histoty.back()

    • 移动到上一个网址

  • history.forward()

    • 移动到下一个网址

  • history.go(num)

    • 指定网址

  • history.pushState(state,title,url)

    • 添加一条记录

  • history.replaceState(state,title,url)

popstate事件

  • history对象出现变化都会触发该事件
